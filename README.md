### 常用知识解释

语句覆盖率和行覆盖率在定义上十分接近，很多资料都会将其画等号。在实际开发中，如果一行代码中只有一条可执行语句，此时语句覆盖率等同于行覆盖率。如果一行代码中有多条可执行语句，并且有存在执行语句执行不到的情况下，语句覆盖率和行覆盖率就不相同。
